Integração do sistema de saúde com o transporte público: API para bloqueio do cartão transporte de pessoas em isolamento social


Esta aplicação permite cadastrar, consultar, editar e deletar registros de pessoas em sistemas fictícios de saúde pública e do transporte coletivo de passageiros a partir de uma API conectada ao banco de dados NoSQL MongoDB.
Conforme as informações presentes nos sistemas, o usuário terá seu acesso ao transporte público permitido ou bloqueado. Para realizar essa verificação, uma página que simula a passagem do cartão transporte foi criada.
A API foi desenvolvida para ser executada sobre o Node.js, utilizando as linguagens typescript/javascript e as dependências fetch, mongoose e express.
